# Fakemon Website

> This is a streamlit repository. Live Website at https://fakemon.streamlit.app

### How to Set Up Streamlit

1. Create a Streamlit Account
2. Create a New Project and Link Repository
3. Add API Keys (For Database and OpenAI)

### Managing Your OpenAI API Key

1. Create/Login to your OpenAI Account
2. Navigate to the API Manager
3. Add Credits to your Account
4. Get API Keys

### How to Create Supabase DB

1. Create/Login to your Supabase Account
2. Create new Project
3. Create new Table with Pokemon Datatypes
4. Add Database Policies for `SELECT` and `INSERT`
